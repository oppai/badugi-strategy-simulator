# 対 J 残し（R1=11, R2=11, R3=11）における最適戦略レポート

本レポートでは、バドゥーギにおいてすべてのドローステップで「J（ジャック、11）以下」を残すという相手の戦略（R1=11, R2=11, R3=11）に対して、最も有効かつ有意に勝ち越すことができる戦略について考察します。

## 1. 基本となる最適戦略：段階的な緩和

相手が全てのドローでJ以下を残す場合、相手の戦略は比較的ルーズであり、ある程度弱い手（例えば T, J のバドゥーギ）でもスタンドパットし、ドローを終えてしまいます。
これに対し、少しタイトな基準でドローを進め、最終ラウンドで相手のレンジに合わせる戦略が有効であることがシミュレーションにより確認されました。

具体的には、**「ラウンド1は9以下、ラウンド2は10以下、ラウンド3は11以下（R1=9, R2=10, R3=11）」** を目指す戦略が、ランダムな初期手札から開始した場合に最も高く安定した勝率（約1.6%のエッジ）をもたらします。

### 基本戦略のシミュレーション結果（ランダム手札）

相手の戦略を固定し（P2: 11, 11, 11）、様々な戦略（P1）を用いて100万回のモンテカルロシミュレーションを実行した結果、以下の勝率が得られました（試行を複数回行い、約0.1%の分散を超えた有意な差として確認）。

| P1 戦略 (R1, R2, R3) | P1 勝率 | P2 勝率 | 引き分け率 |
| :--- | :--- | :--- | :--- |
| **9, 10, 11** | **50.68%** | **49.11%** | **0.21%** |
| 9, 10, 12 | 50.64% | 49.16% | 0.20% |
| 9, 9, 9 | 50.57% | 49.23% | 0.20% |
| 10, 10, 10 | 50.38% | 49.44% | 0.18% |
| 11, 11, 11 | 49.86% | 49.98% | 0.16% |

序盤からJで妥協してしまうと相手と同等レベルの弱いバドゥーギを構成してしまいますが、R1=9、R2=10と引き締めることで、相手のJ・Tバドゥーギに対して確実に勝てる手を目指し、最終ラウンドではバドゥーギ完成を優先してJまで妥協するアプローチが最強となります。

---

## 2. 相手のアクション（チェンジ枚数）に応じた条件付き戦略

相手のチェンジ枚数（Discard Count）をもとに「相手がスタンドパット（完成）したか」「まだドローしているか」で目標ランクを変化させる高度な条件付き戦略も調査しました。

しかし、シミュレーション（100万回）の結果、**条件付き戦略（例: 相手が完成した場合はR2/R3で目標を10に引き締め、ドローしているなら11や12に緩める）を採用した場合の勝率は `50.56% ~ 50.62%` に留まりました。**
これは、バドゥーギの性質上「相手がJで完成しているなら、少し強めの10以下を狙う」としても、そもそもバドゥーギが完成する確率自体が下がるリスクのほうが大きいため、固定戦略（9, 10, 11）の方が分散が少なく安定して強い結果となりました。

---

## 3. 初期手札に依存する最適戦略の変化

手札がランダムに配られた直後（初期手札）の状況によっては、最適なドロー基準が劇的に変化することが分かりました。50万回のシミュレーションにより、初期手札ごとの最適解を検証しました。

### A. 非常に弱い3カード（例: 9s, Th, Jd, 余りTc）からのスタート
すでに「9, T, J」などの弱い3枚が揃っている場合、基本戦略である「9, 10, 11」ではJを残してしまい、相手と同等以下の弱い手で勝負することになります。
このような状況では、**あえて非常にタイトな戦略（R1=7, R2=8, R3=9）を取ることで勝率が大幅に向上** します。

| P1 戦略 | P1 勝率 | P2 勝率 | 備考 |
| :--- | :--- | :--- | :--- |
| **7, 8, 9** (タイト) | **39.93%** | 60.07% | 不利な手札だが被害を抑え、まくりの可能性を最大化 |
| 9, 10, 11 (基本戦略) | 36.76% | 63.24% | そのまま弱い手を完成させると負けやすい |
| 11, 11, 11 (相手と同じ) | 36.54% | 63.46% | 相手と同等 |

弱い手札からスタートした場合は、相手の「J残し」の弱点を突くために、思い切って7や8以下の強いカードを引きに行く（不要な高いカードを捨てる）方が、約3.2%も勝率が高くなります。

### B. バラバラの不要な手札（例: Ks, Kc, Kh, Kd 等）からのスタート
バドゥーギの目が薄い絶望的な手札からスタートした場合、基本戦略（9, 10, 11）とタイト戦略（7, 8, 9）、ルーズ戦略（11, 11, 11）はいずれも勝率40.5%〜41.3%に収束します。極端に緩い戦略（13, 13, 13）を取らない限り、大きな差は出ません。

---

## 結論

すべてのドローステップで「J」を残す（R1=11, R2=11, R3=11）戦略の相手に対しては、以下のプレイングが最適です。

1. **基本は「序盤を少しタイトに引き締め、終盤で基準を緩める」戦略（R1=9, R2=10, R3=11）が最も有効** であり、相手の戦略に対して約1.6%の有意なエッジをもたらします。
2. 相手のチェンジ枚数に依存した複雑な条件付けを行うよりも、自身のハンド構築を優先する（固定の緩和戦略を取る）方が勝率が安定します。
3. **初期手札がすでに「9, T, J」などの弱い3枚で構成されている場合のみ、戦略を大きくタイト（R1=7, R2=8, R3=9 など）にシフトさせる** ことで、相手のJバドゥーギを逆転できる確率が約3.2%向上します。
